# 1. 웹 프론트엔드 상세 명세 (Web Frontend)

- **위치:** `hachilles/web/src/`
- **개요:** AI 에이전트 하네스 품질을 시각화하고 대시보드를 제공하는 React 기반의 SPA(Single Page Application).
- **기술 스택:**
    - React + TypeScript + Vite (빌드 도구)
    - Recharts (데이터 시각화 - scan history, trend charts)
    - CSS: Tailwind CSS 기반의 스타일링
- **주요 기능:**
    - **대시보드:** S~D 등급별 점수 시각화 및 트렌드 차트.
    - **스캔 이력:** 로컬 SQLite 기반의 스캔 히스토리 조회 및 비교.
    - **상태 관리:** React context를 활용한 분석 상태 관리.
